# ChatGPT Userscripts

Two independent browser-user-script projects for improving a ChatGPT workflow.

## Projects

- [`crash-reload-userscript`](./crash-reload-userscript) detects a stale or crashed tab and attempts recovery.
- [`tampermonkey-main-flow`](./tampermonkey-main-flow) adds queueing, sequencing, prompt navigation, and rate-limit-aware sending controls.

Install the scripts from either subdirectory in Tampermonkey, Violentmonkey, or another compatible userscript manager. Each subdirectory contains its own behavior notes and limitations.
